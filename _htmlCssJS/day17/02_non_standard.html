비표준 속성 (non-standard attribute)

  • html 태그의 속성이 제한적이므로 데이터를 관리하는 차원에서
    개발자들에게 직접 속성을 만들어 사용할 수 있는 방법을 제공
  • 너무 과도하게 사용하면 개발 효율과 가독성이 저하될 수 밖에 없으므로, 적당히 사용할 것
  • 태그 속성의 이름을 data-변수명="${값}"
    - 뒤에 변수명을 사용, value 값을 인지할 수 있도록 설계되어 있음
  • 값을 읽어올 경우 : dataset.변수명 => 값이 리턴됨
    dataset property를 사용하여 읽어들임
  • 변수명에 대문자 허용 X , 전부 소문자로 쓸 것 ===> but 대문자로 썼을 때 에러가 나지는 않음
